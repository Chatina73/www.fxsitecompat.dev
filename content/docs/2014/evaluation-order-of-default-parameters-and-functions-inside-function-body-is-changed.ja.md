Firefox 15 で実装された [デフォルト引数](https://developer.mozilla.org/docs/Web/JavaScript/Reference/default_parameters) は、これまで関数本体内の関数の後に評価されていましたが、挙動が変更され、それらの関数の前に評価されるようになりました。そのため、デフォルト引数からそれらの関数を参照することはできなくなります。
