The implementation of the `displaystyle` attribute has been refactored to adhere to the spec. `displaystyle` on the [`<mtable>`](https://developer.mozilla.org/en-US/docs/Web/MathML/Element/mtable) element is no longer inherited from the root [`<math>`](https://developer.mozilla.org/en-US/docs/Web/MathML/Element/math) element. If the attribute is not present, it is considered as `false`, the default value of `displaystyle`. See the [default stylesheet](https://dxr.mozilla.org/mozilla-release/source/layout/mathml/mathml.css) for how it works.
